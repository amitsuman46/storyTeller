STEP 1: CREATE .env FILE
-------------------------
Create a file named ".env" in the project root with these contents:

VITE_SECRET_KEY=mySecretKey123
VITE_GEMINI_API_KEY=your-gemini-api-key-here

Replace "your-gemini-api-key-here" with your actual Gemini API key.
Get your API key from: https://makersuite.google.com/app/apikey


STEP 2: INSTALL DEPENDENCIES
-----------------------------
Open terminal in this folder and run:

npm install


STEP 3: START THE APP
----------------------
Run:

npm run dev

Then open: http://localhost:3000


STEP 4: LOGIN
-------------
Enter the secret key you set in .env file (default: mySecretKey123)


STEP 5: ENJOY!
--------------
- Choose a genre
- Generate a story
- Click Play to hear it with word highlighting
